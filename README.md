A collection of cybersecurity and penetration testing cheatsheets created during my journey studying offensive security and preparing for the **CPTS (Certified Penetration Testing Specialist)** certification from Hack The Box.

This repository contains practical notes, commands, and quick references that I use during labs, machines, and real pentesting scenarios.

The goal is to build a personal knowledge base while also helping other learners in cybersecurity.
